## 🚀 Featured Projects

### [PhotoBoss (C++ / Qt)](https://github.com/maximoh-mmo/photoboss)
A high-performance desktop application for managing massive photo libraries using perceptual hashing and multithreaded pipelines.
- **Differentiator**: Implemented a lock-free producer-consumer pipeline that reduced processing time for 50k images from minutes to seconds via SQLite caching.
- **Tech**: C++, Qt, SQLite, Multi-threaded IO.

### [Labyrinth (C# / Unity)](https://github.com/maximoh-mmo/labyrinth)
A digital recreation of the classic "Crazy Labyrinth" board game, used as a playground for procedural generation.
- **Differentiator**: Built a robust **Constraint Solver** from the ground up to handle valid tile placement and dynamic grid shifting.
- **Tech**: C#, Unity, Procedural Algorithms.

### [Trippy Stargnome (Unreal Engine 5)](https://github.com/maximoh-mmo/trippy-gnome)
A psychedelic 3D rail shooter developed as a student project.
- **Role**: Sole Programmer.
- **Differentiator**: Engineered a fully editable waypoint-based rail system and dynamic enemy spawning logic used by the design team to iterate levels rapidly.
- **Tech**: C++, Unreal Engine 5.

### [Banana](https://github.com/maximoh-mmo/banana)
A focused technical exploration into specific gameplay mechanics and system architecture.
